# Ejecutar en desarrollo

1. Clonar el repositorio
```
git clone https://github.com/dromerof/nest-pokedex.git
```
2. Ejecutar 
``` 
yarn install
``` 
3. Tener Nest CLI instalado
``` 
npm i -g @nestjs/cli
``` 
4. Levantar la base de datos
``` 
docker-compose up -d
``` 

## Stack usado  
* TypeScript
* NestJS
* MongoDB
* Docker
